# Chrome Web Store Listing Draft

## Name

YouTube Subscriptions Fixer

## Short Description

Clean up YouTube Subscriptions by hiding Shorts, removing inserted sections, and showing four videos per row.

## Detailed Description

YouTube Subscriptions Fixer keeps the YouTube Subscriptions page focused on videos from the channels you follow.

Features:

- Hide Shorts shelves, sidebar links, and Shorts video tiles
- Remove inserted subscription sections such as Newest and Most relevant
- Display subscription videos in a four-column grid
- Dim watched videos so new items are easier to scan
- Keep YouTube Home working normally

The extension only runs on YouTube and stores preferences with Chrome storage. It does not use analytics, ads, remote code, or external servers.

## Single Purpose

Customize the YouTube Subscriptions feed display.
